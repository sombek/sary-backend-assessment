## Getting Started

#### Install yarn:
```js
npm install -g yarn
```

#### Install dependencies:
```sh
yarn
```

#### Start server:
```sh
# Ensure that a PostgreSQL database is running on your localhost

# Start server
yarn start

```

#### Tests:
```sh
# Run tests written in ES6
yarn test
```

#### Docker Compose:

```sh
# Run:
docker-compose up
```
